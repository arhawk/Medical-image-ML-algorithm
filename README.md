# Medical-image-ML-algorithm

COMP5318 Assignment 2 project for medical image classification.

## Contents

- `Medical Image ML algoithm.ipynb` - main notebook
- `report.pdf` - written report
- `Assignment2Data/` - training and test arrays used by the notebook
- `keras_tuning_results/` - saved hyperparameter tuning outputs

## Dependencies

- Python 3
- `numpy`
- `pandas`
- `scikit-learn`
- `tensorflow`
- `keras-tuner`
- `matplotlib`
- `seaborn`

## Run

Open the notebook from this repository root so the relative paths to `Assignment2Data/` and `keras_tuning_results/` resolve correctly.
